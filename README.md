# Dashboard Management System Using Python Flask

A simple dashboard project built using Python Flask, HTML, CSS and SQLite.

## Features
- User Registration & Login
- Dashboard Page
- Session Management
- Password Hashing
- SQLite Database

## Technologies Used
- Python
- Flask
- HTML
- SQLite

## Installation

```bash
git clone https://github.com/karangobade/Dashboard-Management-System-Using-Python-Flask.git
cd Dashboard-Management-System-Using-Python-Flask
pip install flask werkzeug
python app.py
```

## Project Structure

```bash
├── app.py
├── templates/
└── README.md
```

## Run Project

Open in browser:

```bash
http://127.0.0.1:5000
```
